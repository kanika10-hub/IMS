## Inventory Management System

## Overview
A full-stack inventory management application designed to manage products efficiently with real-time stock updates and optimized database performance.

---

 Features
- Real-time stock tracking
- CRUD operations (Create, Read, Update, Delete)
- Data validation for accuracy
- Optimized SQL queries for fast retrieval
- Scalable backend handling 500+ products

---

##  Tech Stack
- Java
- SQL
- HTML
- CSS
- JavaScript

---

##  Performance
- Handles 500+ product records
- Achieved ~30% faster data retrieval

---

## How to Run
1. Clone the repository
2. Open in IDE (IntelliJ / Eclipse)
3. Configure database
4. Run the application
